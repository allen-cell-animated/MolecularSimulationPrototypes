Navigation

To look around: click and drag somewhere other than the menu.
To zoom: use mouse scrollwheel or up/down arrow keys.
To open the menu: click the gray button on the bottom center.

Cycle Diagram

This diagram (left part of menu) shows the 6 possible states for each motor. The transition arrows between states are labeled with the current rate of the transition (units = transitions per second). Each of these rates can be adjusted by clicking and dragging the circular knob within each arrow. The states are defined as follows:

  • Mt-KT = ATP bound, bound to tubulin
  • Mt-KDP = ADP and Pi bound, bound to tubulin
  • KDP = ADP and Pi bound, diffusing freely
  • KD = ADP bound, diffusing freely
  • Mt-KD = ADP bound, bound to tubulin
  • Mt-K = no nucleotide, bound to tubulin

Average Walking Speed

Calculated by dividing the distance travelled by the time passed since start (or restart).

Time per Step

The simulated time that passes in each calculation step of the simulation. Increasing it will increase the probability of transitioning state, since probability of state transition is calculated from the number of events that would theoretically happen in a given time interval.

Steps per Frame

The number of simulation steps for each frame that you can see rendered. It functions like a 'fast forward' button. For example, if you set it to 50, each motor will take 50 random steps for every 1 that you see rendered.

Necklink Length

The length of the unstructured necklinker peptide that connects the C-terminal of the motor domain to the 'hips' (the start of the tropomyosin coiled-coil).

Snapping Speed

Once ATP binds, the hips move to the docked position in front of the bound motor with this speed (units = degrees per real-world second)

Mean Step Size

The average distance a freely diffusing molecule moves in one step. Each step, this distance is randomly sampled from an exponential distribution with this as the mean.
